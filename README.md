# BuddyOS

BuddyOS is a browser-based **student productivity workspace** that combines a Pomodoro focus timer, quick notes with Obsidian-style links, a research dock, and a simple timeline â€“ all in one page.

## Features

- Pomodoro-style **Focus Timer** with focus / short break / long break.
- **Notes** with `[[Note Title]]` links between notes and editable note cards.
- **Research Dock** for saving sources into collections with types and notes.
- **Timeline/History** of completed focus sessions.
- Light/dark theme, readable typography, and a clean dashboard layout.
- Single-page app 
- Works as a static site (GitHub Pages, Netlify, etc.)

## Tech Stack

- HTML, CSS, Vanilla JavaScript
